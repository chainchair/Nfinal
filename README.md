# enFinal

Calculadora estática de notas para estudiantes. Ingresas tus calificaciones y obtienes al instante el acumulado, el promedio, la nota mínima necesaria y la nota recomendada para alcanzar tus objetivos.

## Tecnologías

- HTML
- CSS
- JavaScript vanilla

## Cómo usar

ingresa a la pagina web:

https://enfinal.web.app/

## Funcionamiento

1. Ingresa tus notas en los campos correspondientes.
2. Especifica el porcentaje de cada evaluación.
3. La app calcula automáticamente:
   - **Nota acumulada**: lo que llevas hasta el momento.
   - **Promedio actual**: tu rendimiento general.
   - **Nota mínima**: lo mínimo que necesitas obtener para aprobar.
   - **Nota recomendada**: lo que deberías obtener para asegurar un buen resultado final.

## Funcionalidades

- Cálculo de notas con ponderación
- Promedio actualizado en tiempo real
- Proyección de nota mínima para aprobar
- Sugerencia de nota recomendada
- Diseño responsive y amigable
- No recopila datos, sin publicidad ni rastreadores
